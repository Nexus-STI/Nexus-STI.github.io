# NEXUS

Site web de présentation pour le projet NEXUS.

Ce dépôt contient des pages HTML, des feuilles de style CSS et des scripts JavaScript pour le site du projet.

## Structure

- `*.html` : pages du site
- `*.css` : styles associés
- `*.js` : scripts client
- `image/` : ressources graphiques

## Développement

Ouvre les fichiers HTML dans un navigateur pour voir les pages.
